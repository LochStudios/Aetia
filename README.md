# Aetia

Aetia Talent Agency is a creator-focused company built to support talent growth through structured communication, business operations, and long-term partnership management.

This website is Aetia’s operational platform. It is used to manage:

- creator and account onboarding
- professional communication routing and message handling
- contracts and billing workflows
- document management
- internal admin operations, including email and SMS processes

## What Aetia Is

Aetia is a subsidiary of LochStudios and operates as a professional talent agency for creators. The brand focuses on helping creators access opportunities while keeping communication and operations organized, secure, and accountable.

## Purpose of This Repository

This GitHub repository is for Aetia’s company website and internal platform management.

The code is publicly visible for transparency and review.

It is **not** intended for cloning, reuse as a template, or redistribution as an open-source product.
